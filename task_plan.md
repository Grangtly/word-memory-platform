## 已做决策
| 决策 | 理由 |
|------|------|
| 先用 JSON 文件存储 | 快速跑通流程，无需安装数据库 |
| JWT 认证 | 无状态，适合前后端分离 |
| Vite 代理解决跨域 | 开发环境简单，无需额外配置 CORS |
