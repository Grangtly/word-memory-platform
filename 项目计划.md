## 实际踩坑记录

| 坑 | 解决 |
|----|------|
| Wispbyte 文件路径不匹配 | 换 Railway |
| Railway nixpacks $NIXPACKS_PATH | 改用 Dockerfile 构建 |
| railway.toml builder 值小写 | DOCKERFILE 必须大写 |
| Express 5 app.get('*') 崩溃 | 改用正则匹配 |
| Atlas 连接被拒 | Network Access 加 0.0.0.0/0 |
| URI 参数解析错误 | 去 query string |
| Railway 冷启动慢 | UptimeRobot 保活 |
