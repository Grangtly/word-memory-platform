# 发现与决策

## 需求
- 基于艾宾浩斯遗忘曲线的自适应单词复习系统
- 用户注册登录 → 每日获取待复习单词 → 标记记住/没记住 → 算法调度下次复习时间
- 先用 JSON 文件存储，后升级 MongoDB

## 技术决策
| 决策 | 理由 |
|------|------|
| Express 4.x | 最流行的 Node.js 框架，生态成熟 |
| JWT (jsonwebtoken) | 无状态认证，适合 API 服务 |
| bcryptjs | 纯 JS 实现，无需编译，跨平台兼容 |
| 复习间隔 9 级 | 5分钟→30天，覆盖短期到长期记忆 |
