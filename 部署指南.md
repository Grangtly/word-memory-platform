# 部署指南

## 前置条件

1. 注册 [GitHub](https://github.com) 账号
2. 注册 [Render](https://render.com) 账号（后端部署）
3. 注册 [Vercel](https://vercel.com) 账号（前端部署）

---

## 步骤 1：推送代码到 GitHub

```bash
# 在项目根目录执行
git remote add origin https://github.com/你的用户名/word-memory-platform.git
git branch -M main
git push -u origin main
```

---

## 步骤 2：部署后端到 Render

1. 打开 [Render Dashboard](https://dashboard.render.com)
2. 点击 **New +** → **Web Service**
3. 连接你的 GitHub 仓库
4. 配置：
   - **Name**: `word-memory-api`
   - **Root Directory**: `server`
   - **Runtime**: Node
   - **Build Command**: `npm install`
   - **Start Command**: `node index.js`
   - **Free Instance**: 选择 Free
5. 点击 **Create Web Service**
6. 等待部署完成，获得后端地址，例如 `https://word-memory-api.onrender.com`

> Render 免费实例 15 分钟无请求会休眠，首次访问可能等待 30-60 秒。

---

## 步骤 3：部署前端到 Vercel

1. 打开 [Vercel](https://vercel.com)
2. 点击 **Add New** → **Project**
3. 导入你的 GitHub 仓库
4. 配置：
   - **Root Directory**: `client`
   - **Framework Preset**: Vite
   - **Build Command**: `npm run build`（默认）
   - **Output Directory**: `dist`（默认）
5. 添加环境变量：
   - `VITE_API_URL` = `https://word-memory-api.onrender.com/api`
6. 点击 **Deploy**
7. 获得前端地址，例如 `https://word-memory.vercel.app`

---

## 步骤 4：验证部署

1. 浏览器打开前端地址
2. 注册一个账号
3. 登录后进入学习页
4. 点击记住/没记住，确认功能正常

---

## 可选：绑定自定义域名

### 前端 (Vercel)
1. Vercel 项目 → Settings → Domains
2. 添加你的域名
3. 在域名商添加 CNAME 记录指向 `cname.vercel-dns.com`

### 后端 (Render)
1. Render 项目 → Settings → Custom Domains
2. 添加你的域名
3. 在域名商添加 CNAME 记录指向 Render 提供的地址
